📖 Quick Start — Read This First

How to load and use the workflow

  1. Install ComfyUI → git clone https://github.com/comfyanonymous/ComfyUI
  2. Install ComfyUI-Manager into ComfyUI/custom_nodes/ → git clone https://github.com/ltdrdata/ComfyUI-Manager
  3. Launch ComfyUI: python main.py --medvram (best for 8GB VRAM)
  4. Open browser at http://127.0.0.1:8188
  5. Click Manager → Install Custom Nodes → install all required nodes listed below
  6. Download models and place them in the correct folders (see Folder Structure below)
  7. Drag the .json file into your ComfyUI browser window to load the workflow
  8. Start with Section 4 (LoRA Usage) — it's the simplest!
⚠️ Only run ONE section at a time to preserve VRAM. Right-click a group and select "Set Group Nodes as Muted" to disable sections you're not using.

💡 VRAM Saving Tips for RTX 3070 Ti (8GB)

🚀 Launch ComfyUI with: python main.py --medvram
Use --lowvram if still getting OOM errors.
📐 Keep resolution at 512×512 or 512×768 for SD 1.5. Never use SDXL without --lowvram.
📦 Keep batch_size = 1 always. VRAM multiplies with batch size.
🎞️ AnimateDiff: max 16 frames at 512×512. Reduce to 8 if needed.
🔧 Enable VAE Tiling in ComfyUI settings (gear icon) for upscaling large images.
🎮 Close Chrome, Discord, games, and other GPU apps before running heavy workflows.
🔄 Only run ONE section at a time. Mute/disable inactive groups.
🧠 Use fp16 ControlNet models — they use ~half the VRAM of full precision.
